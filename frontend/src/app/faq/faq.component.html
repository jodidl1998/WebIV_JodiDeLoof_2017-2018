'Deadlines' is een simpele oplossing om als groep alle deadlines op 1 plaats bij te houden.

Na het registreren kan je een 'Classroom' aanmaken door op Classroom aanmaken te klikken.

Na het aanmaken van de Classroom zie je linksboven een 8 cijferige code. Aan de hand van deze code kunnen klasgenoten zichzelf toevoegen.

Je kan een classroom joinen op 2 verschillende manieren.

Je hebt nog geen account

Op de home pagina kan je in het tekstvak een 'Classroom' code ingeven. Door hierna op joinen te klikken wordt je doorgestuurd naar de registreer pagina. Indien je al een account hebt klik je op Login

Na het registreren zal je automatisch worden toegevoegd aan de Classroom.


Je hebt al een account

Op dit moment is het nog niet mogelijk om deel uit te maken van verschillende classrooms. Om van classroom te veranderen klik je op de knop

Verlaten

Hierna kan je jezelf aan de hand van de code toevoegen aan een Classroom door te klikken op de knop

Classroom joinen
